## 📌 Sobre o Projeto

Este projeto consiste na recriação do site institucional da **ETEC Zona Leste**, utilizando **HTML, CSS e PHP**.

A proposta é desenvolver uma versão mais moderna, organizada e funcional do site original, mantendo a identidade visual do **Centro Paula Souza**, mas com melhorias na estrutura, navegação e experiência do usuário.

#

## 🎯 Objetivos

- Criar uma interface mais moderna e intuitiva  
- Melhorar a experiência do usuário (UX)  
- Aplicar organização de código com PHP  
- Desenvolver um site responsivo  
- Utilizar boas práticas de front-end e back-end  

#

## 🚀 Tecnologias Utilizadas

| Tecnologia | Finalidade |
|-----------|------------|
| **HTML5** | Estrutura das páginas |
| **CSS3** | Estilização e layout responsivo |
| **PHP** | Organização e reutilização de componentes |
| **JavaScript** | Interatividade básica |
